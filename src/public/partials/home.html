AngularJs Coding Callenge

You’ll create an AngularJS web application to showcase your Angular and JavaScript skills.

The app will consume a web service and display the results for browsing. You’ll use services,routing, localStorage and everything from the UI, tooling will be evaluated. When complete, you’ll upload your code to a GitHub public repository and share the URL for review.

Objective:

Using the latest version of AngularJS (1.4.x) and Bootstrap (AngularUI or CSS), develop an Angular web application to display Food Reports from the USDA National Nutrient Database.

Scope

  • As a user I want to search for a specific food and view the results.
  • As a user I want to select a food to view a "Basic Report" on a new page.
  • As a user I want the food report (Basic Report) to display all nutrients and their values in grams.
  • As a developer I want an angular service to consume the API.
  • As a developer I want the data to be saved and retrieved from html5 localStorage.
  • As a developer I want an API key so I can call the NDB REST API.
